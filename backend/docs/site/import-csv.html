Formato del archivo

  • Archivo CSV UTF-8, separador coma (,)
  • Primera fila: nombres de columnas (encabezados)
  • Una fila = un ítem en una ubicación con su cantidad

Columnas

Columna Obligatorio Descripción Ejemplo
nombre Sí Nombre de la herramienta Llave allen 10mm
marca No Marca Stanley
modelo No Modelo SA10
tipo No Categoría Herramienta
detalle No Notas Juego métrico
armario Sí A00, A01 o A02 A01
estante Sí E01 a E09 (o 1–9) E01
contenedor No C01–C99, B00–B99, H01–H99 o SC. Vacío = suelto en estante C01
cantidad Sí Entero ≥ 0 8

Modos de importación

agregar Suma cantidades si el ítem ya está en la misma ubicación. Crea ítems y ubicaciones nuevas si no existen.

reemplazar Solo con DEMO_MODE=true: borra inventario demo y carga solo lo del CSV.

Desde la aplicación

Iniciá sesión como administrador → menú Importar → subí el CSV o pegá el contenido.

API: POST /api/admin/import/csv con body JSON { "csv": "...", "modo": "agregar" } y header Authorization: Bearer <token>
